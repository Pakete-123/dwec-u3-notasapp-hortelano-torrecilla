# Guía de usuario (esquemática)

- Propósito: gestionar notas con filtros Hoy/Semana/Todas.
- Requisitos: navegador moderno; permitir ventanas emergentes para el Panel.
- Tareas: añadir nota; aplicar filtros; abrir Panel; completar/borrar.
- Preferencias (si las incorporas): tema/tamaño.
- Persistencia: indicar mecanismo elegido y sus implicaciones.
- Problemas comunes: pop-ups bloqueados; almacenamiento deshabilitado; idioma del navegador.

## Pasos

1. **Rellenar el campo "Texto"**

- Este campo se ha creado para añadirle un título a la nota. Se puede poner lo que sea hasta un máximo de 200 carácteres.

2. **Seleccionar Fecha**

- Hay que seleccionar una fecha que no sea anterior a la de hoy para poder mostrar las notas mediante los filtros y saber la fecha "límite" que tiene la nota.

3. **Elegir la prioridad**

- Hay que elegir una prioridad para la nota, que la ordenará según prioridad.

4. **Clicar el botón de "Añadir"**

- Al darle a este botón, se añadirá una nota al gestor de notas.

5. **Botón de Completar**

- Este botón marca la nota como completada, poniéndola de color verde, o quitándole el color si se le vuelve a presionar.

6. **Botón de Editar**

- Este botón permite modificar la información de la nota y guardarla.

7. **Botón de Eliminar**

- Este botón elimina una nota, después de que se confirme la solicitud de eliminación.

8. **Desplegable de Restaurar Versiones**

- En este desplegable se encuentran las últimas 5 versiones del gestor de notas, donde se podrá revertir cualquier cambio relacionado con las notas, como una edición o una eliminación de una nota.

9. **Filtros**

- Existen 3 filtros en el gestor de notas, y se encuentran en la parte superior derecha de la pantalla. El primero es el de "Hoy", que mostrará solamente las notas con la fecha de hoy. El segundo es el de "Semana", que mostrará las notas que estén dentro de un marco de 7 días empezando desde el día siguiente al actual. El último es el de "Todas", que mostrará todas las notas independientemente de la fecha que tengan.

10. **Panel Diario**

- Este botón ubicado en la parte inferior izquierda abrirá una ventana emergente en la que se encontrarán las notas con las fechas del día de hoy.
